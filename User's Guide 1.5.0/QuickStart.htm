KICKS

A transaction processing system for CMS & TSO

 

Follow the steps in the Installation section of this manual to quickly get KICKS installed on your machine.

KICKS works much like CICS. If you want a quick demonstration of this try the sample programs described in the Examples section. For a brief description of how the TSO/KICKS environment is different from CICS see the TSO / CICS / KICKS appendix.

For a checklist to get your own application running under KICKS see the Programming section.

For reference information about KICKS tables and startup see the Configuration and Operation sections.

Finally, for in depth tutorials see the KooKbooK recipes online at http://www.kicksfortso.com/same/KooKbooK/KooKbooK.htm
